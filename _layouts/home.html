This site tests whether it is a suitable replacement for HISCOM's Mediawiki instance. A suitable replacement would include the following features:

  • Markdown (and optionally Asciidoc) source files for pages
  • Automatic update of the website from the Github source
  • Direct editing of source files in Github
  • Site search
  • Look nicer than Mediawiki
  • Site design would cleanly separate work-in-progress from published, final, documents for public use.

The set of terms in the HISPID 6 vocabulary, based on Darwin Core. This vocabulary suprsedes all previous versions of HISPID.
